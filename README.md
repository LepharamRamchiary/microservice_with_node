### Ngrok
- This project uses Ngrok to securely expose local development servers to the internet for testing webhooks, APIs, and external integrations. Simply run Ngrok from the command line to generate a public URL that tunnels traffic directly to the local application.

### Use library for development and testing
- Web Push -> I used Web Push notifications for order creation events for testing and learning purposes.
- Winston -> Implemented Winston logging to record application events, errors, and request activity for debugging, monitoring, and learning purposes.
